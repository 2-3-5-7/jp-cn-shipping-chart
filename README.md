线路可分为两类，包税线和不包税线，这个维度就要基于货值来选线路，包税线的价格更贵

- 税小于 50 免收，多数 0.2 税率，货值要小于 250 (50 / 0.2)，纸片类、毛绒、手办、游戏机和卡带是 0.13 小于 380，混合就要具体算
- 书籍和光盘有免税额，这个邮政和纸飞机线经常用
- 包税：彩虹桥XX、竹蜻蜓 plus、拼邮-任你购、申通-乐酷淘、鱼鹰-docobuy
- 不包税：纸飞机、竹蜻蜓、筋斗云、邮政线路（ems、海运、空运）

另一个维度是重量（有的有体积重），重量越大每克运费越便宜，所以可以多买点然后 🐟 卖掉抵扣运费，重量低运费就不划算，当然拼邮稍微好些

两个维度就可以形成 4 种情况

- 很少量，物品净重 300g 以下，任你购拼邮

1. 便宜且轻，竹蜻蜓，物品类别不能走的话，纸飞机、筋斗云
2. 贵且轻，乐一番包税线。第二考虑，docobuy、乐酷淘，这些很少打折，但运费便宜的
3. 便宜且重。5kg 以内纸飞机，5 kg 以上海运。3kg 以上如果货物便宜，也可以考虑筋斗云
4. 贵且重

- 对于可拆分的，考虑拆分为情况 2 + 3 或 多个情况 3，比如书籍用纸飞机发货，贵的谷子用彩虹桥发货。很多手办和毛绒，分箱后，控制在 380r 以内，发多次纸飞机
- 对于不可拆分的，就要对比走不包税线的税 与 走包税线多花的运费来选择

- 运费不是全部。乐一番商品打折、积分，就相当于节省运费了，尤其对于贵的东西，打九折，就可以抵扣多收取的运费

线路注意

- 乐一番，箱子费，彩虹桥plus有体积重
- 日淘市集
加固费 200 日元，但同时 500 积分抵扣很容易做到（去🐟买）
500g 一档，重量低可能跳档不划算
手续费、合单费用、异形件费用、最低申报
筋斗云没有书籍光盘免税额，而书籍光盘线额外加收 5% 费用
- 挖煤姬，手续费，竹蜻蜓有类别限制
- 任你购，手续费，拼邮有类别限制。拼邮国际段不算箱子重，为了与其它线路比较，假设了 200g 箱子
- 乐酷淘，箱子费
- docobuy，箱子费，最便宜的包税线鱼鹰，2 万日元以内，商品小于 15 个，cd 不能混发，这条线比较慢
- jpgoodbuy，老平台，运费没优势。凤凰线不能合箱。除乐一番外的，即做转运又做代购的平台，但邮政线也不让发 r 本
- 精灵集市，新平台，政策很不稳定（一个月改好几次），客服体验差，各种营销，也运营转运“池袋顺丰”。客观说，申通运费比乐酷淘便宜，但我不推荐
